# 🚗 Streamlit Car Data App

Aplicação interativa desenvolvida em **Python** com **Streamlit**, **Pandas** e **Plotly Express** para visualização de dados de anúncios de vendas de carros.

## 📊 Funcionalidades
- Criar **histograma** da quilometragem (`odometer`) dos veículos.
- Criar **gráfico de dispersão** relacionando preço (`price`) e quilometragem (`odometer`).
- Interface amigável e interativa via Streamlit.
